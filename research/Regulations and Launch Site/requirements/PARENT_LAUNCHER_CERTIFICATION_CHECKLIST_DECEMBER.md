# Parent Launcher Checklist: Certify to Fly a Built ~20 kN·s Hybrid by December

**Roles**
- **Parent** = launcher / Flyer of Record / N₂O custodian / Tripoli-certified adult  
- **Minor (17)** = motor and vehicle **builder** (design, fabricate, document) under parent supervision  

**Target.** Parent legally and under Tripoli rules able to **launch the minor’s built ~20 kN·s experimental hybrid** (N₂O + paraffin) by **December** (this year).  

**Assumption.** Work starts **now (mid-July)**. This schedule is **aggressive but feasible** for **Level 1 → Level 2** plus hybrid/Research Launch prep. **Level 3 is optional** and may not fit before December—**L2 is the usual minimum** to fly research/hybrid motors at a Tripoli Research Launch.

**Not legal advice.** Confirm dates with Florida prefecture calendars, the Research Launch Director, and current Tripoli rules.

**Related.** `TALLAHASSEE_20_kNs_LAUNCH_REQUIREMENTS.md` · `../../N2O_paraffin_burn_rate/SAFETY_PROCEDURES_N2O_PARAFFIN.md`

---

## 1. Success criteria (December)

Parent can check **all** of the following:

- [ ] Tripoli **Senior** member in good standing  
- [ ] **Level 2** certification active (card / HQ processed)  
- [ ] Comfortable with **dual-deploy** electronics and range safety  
- [ ] **N₂O** purchase, storage, remote **fill/dump** practiced with experienced help  
- [ ] Minor’s hybrid **static-fired** successfully; impulse ~20 kN·s documented  
- [ ] Flight vehicle RSO-ready (stability, recovery, checklists)  
- [ ] **Tripoli Research Launch** slot / approval for this research hybrid  
- [ ] **FAA waiver** covers the event (usually via club LD)  
- [ ] Parent listed as **Flyer of Record**; minor listed as builder/owner as allowed  
- [ ] Parent/guardian consents and travel plan locked  

If any box is open in early December, **slip the flight**—do not pad an unproven hybrid.

## 2. Master checklist (parent must complete)

### A. Membership and paperwork

- [ ] Create Tripoli account; join as **Senior** member (18+) — https://www.tripoli.org  
- [ ] Pay dues; save membership number  
- [ ] Read **Unified Safety Code** end-to-end  
- [ ] Read project **SAFETY_PROCEDURES_N2O_PARAFFIN.md**  
- [ ] Minor joins as **Junior** (for TMP skills track; optional but recommended)  
- [ ] Sign minor’s TMP consent when they take the TMP exam  
- [ ] Add Florida prefecture contacts (Tampa / Space Coast / Fort Myers)  
- [ ] Put **all 2025–2026 launch dates** on the calendar (see §3)  

### B. Level 1 certification (parent)

- [ ] Study HPR basics (construction, stability, recovery, safety code)  
- [ ] Choose L1 rocket (sturdy kit or simple scratch; H or I certified motor)  
- [ ] Buy rocket + **certified H or I motor** + recovery supplies from a vendor (you are 18+)  
- [ ] Build rocket; practice packing chute / shear pins if used  
- [ ] Attend at least one launch as observer before cert flight (strongly recommended)  
- [ ] Schedule L1 attempt with Prefect / cert team  
- [ ] Pass RSO inspection; fly and **recover** successfully  
- [ ] Get Universal Certification Form signed; submit to Tripoli HQ per current instructions  
- [ ] Confirm **Level 1** shows on membership record  

### C. Level 2 certification (parent) — required for research hybrid FoR

- [ ] Hold active L1  
- [ ] Take **Level 2 online exam** (Parts A & B); pass at required score (typically 90%)  
- [ ] Print / save L2 exam certificate for the certifying official  
- [ ] Build or buy L2-capable rocket (certified **J, K, or L** motor)  
- [ ] Install and **ground-test** altimeter(s) / dual deploy if used  
- [ ] Schedule L2 flight at a prefecture launch  
- [ ] Present L2 exam certificate + rocket to cert team  
- [ ] Successful flight and recovery  
- [ ] Submit signed cert form to HQ; confirm **Level 2** active  

**Do not attempt the ~20 kN·s research hybrid as FoR until L2 is confirmed.**

### D. Optional Level 3 (stretch — only if ahead of schedule)

- [ ] Select two **TAP** members  
- [ ] Submit L3 design package; get **written approval before building**  
- [ ] Build new L3 rocket; use **certified commercial M/N/O** (not the homemade hybrid)  
- [ ] TAP-witnessed L3 flight  

Skip L3 if it endangers the December hybrid date. **L2 + Research rules** are the critical path.

### E. Hybrid / N₂O competence (parent as custodian)

- [ ] Study Tripoli hybrid / N₂O tank rules (aluminum/composite tank; **remote fill and dump**)  
- [ ] Arrange cylinder account / purchase path in **parent’s name**  
- [ ] Assemble fill/dump panel; leak-check with experienced mentor present first time  
- [ ] Practice cold-flow / fill/dump away from crowds per SOP  
- [ ] Oxidizer-clean fittings; ban petroleum greases on N₂O wetted parts  
- [ ] Written emergency / dump / fire plan for static tests and flight  

### F. Support the minor builder (parent oversight)

- [ ] Weekly design/build review with the minor  
- [ ] Enforce casting and shop safety (no solo hot wax / no solo N₂O)  
- [ ] Require **static fire** campaign before flight authorization  
- [ ] Keep a binder: drawings, sims, static-fire data, checklists, photos  
- [ ] Ensure grain inspection (voids/cracks) before any hot fire  

### G. Research Launch and FAA (December flight)

- [ ] Contact Launch Director **early** (by October) about research hybrid ~20 kN·s  
- [ ] Submit motor/vehicle package when LD requires (design, static data, procedures)  
- [ ] Confirm event is a **Tripoli Research Launch** (not NAR; not cert-only sport launch)  
- [ ] Confirm FAA waiver altitude/dates cover the flight (club LD)  
- [ ] Pre-register; list **parent = Flyer of Record**, Tripoli number on flight card  
- [ ] Pack recovery, tracking, tools, spare shear pins, batteries, checklists  
- [ ] Travel / lodging / weather backup weekend identified  

### H. December go / no-go (parent decision)

Fly only if:

- [ ] L2 active  
- [ ] Static fire data acceptable  
- [ ] LD/RSO accepted the project  
- [ ] Waiver active; weather within rules  
- [ ] Fill/dump and abort paths tested  
- [ ] You are willing to **scrub** on any safety doubt

## 6. Common failure modes (avoid these)

1. Waiting until October to join Tripoli  
2. Skipping observer launches and failing L1 on procedure  
3. Taking L2 exam the morning of the cert flight without study  
4. Treating the homemade hybrid as an L2/L3 **certification** motor (not allowed)  
5. First N₂O use on flight day  
6. No static fire “because the sim looks fine”  
7. Assuming a normal sport launch will allow a research hybrid  
8. Parent not yet L2 but listed as FoR for the hybrid
